# quiz-app

# Develop/Build Instructions

## Front End

- Develop: ```yarn start```
- Build: ```yarn build```

# Run Instructions

- Build front end and copy `build/` into `server/static/`.
- Run: ```pip install --no-cache-dir -r requirements.txt && cd server/ && python -m flask run```
- URL: http://127.0.0.1:5000/static/index.html
